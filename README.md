# Local Setup

## Computer Set Up

- Install Node.js (*latest stable version) & NPM
- Install typescript globally
---

## Commands
- `npm i -g typescript`
- `npm run dev` - To run the project


### To use the API

- host:port/api/transaction/:transactionId/:confidenceLevel
